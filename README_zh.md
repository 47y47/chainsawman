# 电锯人待办 (Chainsaw Man Todo)

《电锯人》主题的 Windows 桌面待办应用，基于 Electron + Vanilla JS + SQLite。

## 功能特性

### 核心待办管理
- 添加、编辑、删除、拖拽排序待办项
- 三种类型：一般任务、每日契约（每日重置）、每周讨伐（周一重置）
- 基于 ISO 周的导航系统，支持前后翻阅
- 可设置提醒时间，通过 Windows 原生通知推送

### 拉环完成机制
- 拖动三角形拉环"启动电锯"，完成待办
- SVG 动态线缆跟随鼠标，张力变色（灰→红）
- 未达阈值松手弹簧回弹动画

### 评分系统
- 10 格雏菊方框，0-10 分评分
- 悬停预览，点击锁定确认

### 波奇塔桌面宠物
- 8 帧走路精灵动画，左右巡逻
- 点击跳跃（连续点击可叠加高度）
- **战斗系统**：完成待办后，波奇塔切换战斗形态
  - 缓动飞向待办位置
  - 4 次渐进裂痕锯击 + 1 次最终撞碎
  - SVG 裂痕特效、火花粒子、碎片爆破
  - 完成后的待办被视觉摧毁并从列表移除

### 音效（Web Audio API）
- 4 段《电锯人》原片音效
- 拉环时播放启动音效 → 待机轰鸣循环
- 释放后播放战斗 BGM
- 每次锯击触发撞击音效叠加在 BGM 上
- 预解码 AudioBuffer，零延迟播放

### 每周总结
- 平均分 + 玛奇玛评价台词（0-10 分共 11 档）
- 11 张 Q 版玛奇玛插图对应各分数段
- 统计：已完成/总数、最高分、最低分

### 设置
- 全局静音开关
- 开机自启动（HKCU 注册表，无需管理员权限）

### 系统托盘
- 最小化到托盘，双击恢复
- 右键托盘菜单：显示窗口 / 退出

## 快捷键

| 按键 | 功能 |
|------|------|
| `Ctrl+N` | 新建待办 |
| `Escape` | 关闭所有弹窗/菜单 |

## 技术栈

- **Electron** 35 — 无边框透明窗口
- **Vanilla JS** — 无框架
- **sql.js** — 纯 JS SQLite（WASM）
- **Web Audio API** — 零延迟音频
- **sharp + ffmpeg-static** — 图片/视频处理（开发工具）

## 项目结构

```
chainsawman/
├── main.js              # Electron 主进程
├── preload.js           # contextBridge IPC 桥接
├── src/
│   ├── index.html       # 主窗口
│   ├── main.js          # 渲染进程：UI 逻辑、音频、拉环、评分
│   ├── pet.js           # 波奇塔桌宠
│   └── styles/
│       ├── main.css     # 应用样式
│       └── pet.css      # 桌宠样式
├── assets/
│   ├── audio/           # 音效文件（mp3）
│   └── images/          # 波奇塔精灵图、玛奇玛图片
├── scripts/             # 图片/视频生成工具（Python + Node.js）
└── docs/                # 需求文档、技术规范、设计文档
```

## 开发

```bash
# 安装依赖
npm install

# 开发模式运行
npm start

# 打包 Windows 安装包
npx electron-builder --win
```
