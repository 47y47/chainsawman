# 设计规范 — 电锯人待办事项

## 设计原则
- **简洁第一**：文字清晰可读，不花哨
- **实用优先**：操作流畅，信息密度适中
- **点缀而非堆砌**：电锯人元素在不干扰功能的前提下融入

## 配色方案
| 角色 | 色值 | 用途 |
|------|------|------|
| 主背景 | `#1a1a1a` | 窗口背景 |
| 次级背景 | `#2a2a2a` | 待办卡片、面板 |
| 边框 | `#3a3a3a` | 分割线、边框 |
| 主文字 | `#e0e0e0` | 正文 |
| 次级文字 | `#888` | 辅助信息 |
| 强调色 | `#c41e3a` | 电锯人红，用于重要按钮 |
| 完成色 | `#8b0000` | 深红/血色，已完成的提示 |
| 拉绳色 | `#4a4a4a` | 橡胶灰 |
| 手柄色 | `#d4a574` | 塑料米色 |

## 字体
- 中文：`"Microsoft YaHei", "PingFang SC", sans-serif`
- 等宽/数字：`"Consolas", monospace`
- 字号：正文 14px，标题 18px，辅助 12px

## 窗口
- 默认尺寸：420×620px
- 最小尺寸：360×480px
- 可调整大小
- 圆角窗口（Windows 11 风格）

## 组件规格

### 待办项
- 高度：48px
- 左侧拉绳手柄：32×32px 圆形区域
- 文字区域：flex-grow
- 性质标签：小号 pill 形状（日常蓝、周常绿、一般灰）
- 已完成：黑色血迹 SVG 覆盖层

### 拉绳手柄
- 圆形，直径 32px
- 内圈齿轮纹理（简洁 SVG）
- 绳子：2px 宽灰色线条，从手柄底部向下延伸
- 拖动时绳子拉伸，颜色从灰渐变到深红
- 到达阈值时手柄外圈发光（`box-shadow` 红色辉光）

### 评分方格
- 10 个 28×28px 方格，间距 6px，水平排列
- 未选中：边框 `#555`，背景透明
- 选中（雏菊）：边框 `#f5a623`，背景淡黄，内部雏菊 emoji 🌼 或 SVG
- hover 时预览效果

### 每周总结
- 居中布局，大面积留白
- 上方：大号平均分数字
- 中央：Q 版玛奇玛插图（约 200×200px）
- 下方：台词文字（斜体，居中）
- 底部：补充统计（完成数、最高/最低分）

## 动画参数
| 动画 | 时长 | 缓动 |
|------|------|------|
| 拉绳拉伸 | 实时跟随鼠标 | - |
| 拉绳回弹 | 300ms | ease-out |

| 血迹出现 | 400ms | ease-in |
| 评分方格填充 | 200ms/格 | ease |
| 玛奇玛切换 | 300ms | ease |
