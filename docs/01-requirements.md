# 需求规格 — 电锯人待办事项

## 项目背景
面向电锯人粉丝的 Windows 桌面待办事项应用。以"拉引擎启动绳完成待办"为核心交互，融入电锯人世界观的视觉和音效元素，兼顾实用性与趣味性。

## 功能需求

### F1 按周管理待办
- 以自然周（周一~周日）为单位组织待办
- 默认显示本周，可切换到历史周或未来周
- 支持跨周查看

### F2 待办 CRUD
- **添加**：弹窗输入标题、选择性质、可选提醒时间
- **编辑**：右键菜单修改内容
- **删除**：右键菜单删除（需确认？暂不需要）

### F3 待办性质
| 类型 | 行为 |
|------|------|
| 一般 | 完成后永久划掉，不再出现 |
| 日常 | 完成后当天保留（划掉状态），次日 0:00 自动重置为未完成 |
| 周常 | 完成后当周保留（划掉状态），下周一 0:00 自动重置为未完成 |

### F4 提醒时间
- 添加待办时可选设置提醒时间
- 到时间后弹出系统通知（tauri-plugin-notification）
- 不需要截止时间

### F5 拉绳完成机制
- 待办左侧有拉绳手柄按钮（引擎启动绳样式）
- 鼠标按下手柄 → 向下拖动约 100px →
- 拖动中窗口轻微震动
- 到位松手 → "咔——嗡！！"引擎空转音效 → 拉绳回弹动画 → 进入评分
- 未到位松手 → 仅回弹，不触发完成

### F6 黑色血迹删除线
- 已完成的待办文字用黑色血迹/涂抹风格覆盖
- 不是普通 CSS line-through，是不规则黑色涂抹效果

### F7 评分系统
- 每次完成触发评分界面
- 0-10 分，10 个方格框
- 选中 x 分 → 前 x 个方格显示雏菊 🌼
- 分数写入数据库

### F8 每周总结
- 独立按钮进入总结视图
- 显示本周平均分（已完成的待办）
- 根据平均分（四舍五入）显示 Q 版玛奇玛 + 对应台词
- 满分周（平均分=10）触发彩蛋台词
- 额外统计：完成数、最高分、最低分

### F9 系统行为
- 关闭按钮 → 最小化到系统托盘
- 托盘右键菜单：显示窗口 / 退出
- 开机自启动（启动文件夹）
- 设置面板：静音开关

## 非功能需求
- 简洁实用 UI，不影响文字可读性
- 电锯人元素点到为止
- 单机运行，无网络依赖
- 本地 SQLite 存储
